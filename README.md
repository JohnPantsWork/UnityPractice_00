# UnityPractice_00_desertKnight
橫向卷軸動作遊戲，透過Unity 與 基礎c#建置而成

1. 玩家可以透過鍵盤控制角色跳躍、防禦、移動與攻擊。
2. 武器可以進行切換。
3. 裝備會隨著人物受傷而剝落。
